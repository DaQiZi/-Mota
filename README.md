# -Mota

## 魔塔

核心功能点：
1. 存读档
2. 地图模块的设计
3. 人物走动以及走动过程中和周围物体的交互

特点：
1. 游戏内容基于配置文件，可以灵活配置
2. 可以使用Tile map为游戏编辑地图
